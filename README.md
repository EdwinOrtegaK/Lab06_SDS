# Red Teaming LLM Applications

Este repositorio contiene los notebooks trabajados durante el curso 
"Red Teaming LLM Applications" de DeepLearning.AI.

Para cada sesión con código se agregó al menos una variación propia de ataque,
incluyendo el prompt utilizado, la respuesta obtenida del LLM y un breve análisis
del resultado.
